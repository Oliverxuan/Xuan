# 玄 UI 组件库

## 完美支持 Vue3 和 TS 使用 vite 做到更快速度，更小体积。

Xuan UI component library perfectly fits vue3 and typescript for development

## 开始

### 安装所需依赖

yarn

### 启动服务

yarn dev

### 服务地址

http://localhost:3000/#/

### 打包

yarn build
